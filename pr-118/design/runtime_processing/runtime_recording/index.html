    Runtime recording#

    A LTTng session has to be started before running the application.

    CARETは途中からの計測も可能です。

    これにより、1分間慣らしで動かした後の1分間などの評価が行いやすいようになっています。

    ここでは、Runtime recordingの設計について説明します。 使い方については別途資料を参照ください

    $ export LD_PRELOAD= # 事前準備
$ ros2 launch # 計測対象の実行

$ # 別ターミナルにて
$ ros2 caret record # 計測の開始, 事前にlttng sessionを起動する必要もない。
Ctrl-C（記録終了）

    Overview#

    Tracepointで説明している通り、はランタイムではトレースポイントを極力減らすため、メタ情報は初期化時に保存するようになっています。 そのため、単にLTtngセッションを途中から　起動するだけでは、計測結果が何ノードのコールバックか特定できません。

    CARETは途中からの記録を可能にするために、初期化時の情報をメモリに格納しておき、計測開始後にトレースデータに保存させています。

    課題

    • 途中からの計測対応
    • 起動直後でのロスト抑制 軟骨伝導 これらを解決するため方針として以下を進める。
    • 初期化関連のトレースポイントはメモリ上に保存させる
    • 初期化関連のトレースポイントと計測関連のトレースポイントの記録はタイミングを分ける

    Topic#

    caret_tracing#

    Info

    他の方法として、初期化関連のトレースポイントはblockモードで記録して、 計測関連のトレースポイントはdiscardモードで記録すれば、初期化関連のトレースポイントは確実に保存される。 ロストの発生自体はこれで抑えられる可能性はあるが、今回はcaret側で改善を行うこととした。

    TraceNodeの状態#

    TraceNodeは以下の４つの状態を持つ。

    • 待機状態(WAIT)
    • 計測準備状態(PREPARE)
    • 計測状態(RECORD)
    • 開始直後の計測状態(init_record)
      • これまでの先のlttngセッションを起動させてく利用方法でも使えるようにする、互換性のための状態。
      • 将来的には無くしても良い？

    代表的な状態遷移は

    1. init_record
    2. WAIT
    3. PREPARE
    4. RECORD と遷移する。

    今回の提案では、以下のようにlttng sessionを起動した後に「記録開始トピック」をpublishして、実際に記録を開始させる。

    利用方法の互換性を保つために、事前にsessionを起動するケースでは、記録開始トピックのpublishもせずに記録を開始できるようにしておく。

    主な状態遷移を示す。

    各状態での全イベント（start_recording/end_recording）に対する状態遷移を記述したものを以下に示す。

    開始直後の計測状態(init_record)#

    計測中の状態。 全トレースポイントが同期的に記録される。 静的なトレースポイントはメモリ上に保存する。

    待機状態(WAIT)#

    静的なトレースポイントの保存のみを行う。 「/caret/start_record」トピックの受信で「PREPARE」へ遷移

    計測準備状態(PREPARE)#

    メモリ上に保存した静的なトレースポイントの記録のみを行う。 この際のトレースポイント記録の周波数は任意に設定できること。 計測は行わない。

    計測状態(RECORD)#

    計測中の状態。 全トレースポイントが同期的に記録される。 静的なトレースポイントはメモリ上に保存する。

    ros2caret#

    ros2 traceコマンドに加えて、途中から計測の機能追加に伴い増えるCLI操作の簡略化

    ros2 caret record の中身では、以下のターミナル２とターミナル３の作業が行われている。

    ターミナル1
export LD_PRELOAD= # 事前準備
ros2 launch # 計測対象の実行

ターミナル2
ros2 trace -k -u "ros2*" # 計測の開始

ターミナル3
ros2 topic pub /caret/start caret_msgs/msg/start
ros2 topic pub /caret/stop caret_msgs/msg/stop

ターミナル2にて Ctrl-C# 計測の終了
